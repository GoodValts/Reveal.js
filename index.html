The fr unit

fr - fraction of the available space in the grid container.

CSS:


						.wrapper {
							display: grid;
							grid-template-columns: 1fr 1fr 1fr;
						}

Repeat() notation

CSS:


						.wrapper {
							display: grid;
							grid-template-columns: 1fr 1fr 1fr 5fr;
						}
					

						.wrapper {

Controlling the order


							.wrapper {
								display: grid;
								grid-template-columns: repeat(3, 1fr);
								grid-auto-rows: 100px;
							}
							  
							.box1 {
								grid-column-start: 1;
								grid-column-end: 4;
								grid-row-start: 1;
								grid-row-end: 3;
								z-index: 2;
							}
							  
							.box2 {
								grid-column-start: 1;
								grid-row-start: 2;
								grid-row-end: 4;
								z-index: 1;
